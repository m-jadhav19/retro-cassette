n# Retro Cassette

A nostalgic music discovery app that brings back the magic of 2000s mixtapes. Search for music using AI, generate cassette tapes, and play them in your Walkman.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env.local` file with your Gemini API key:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

## Usage

- Type a music vibe or genre in the search box
- Click "CREATE!" to generate a mixtape
